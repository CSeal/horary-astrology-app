## Description (max 4000 chars)

```
Задайте конкретный вопрос. Получите настоящий ответ — за секунды.

Hora использует классический метод хорарной астрологии (Уильям Лилли, дома Региомонтана) для построения карты в момент задания вопроса и интерпретирует её с помощью ИИ на понятном языке. Время рождения не нужно. Знания астрологии не требуются.

КАК ЭТО РАБОТАЕТ

1. Введите ваш искренний, конкретный вопрос
2. Приложение фиксирует точный момент и ваше местоположение
3. Хорарная карта рассчитывается по классической методике (Swiss Ephemeris DE431)
4. ИИ интерпретирует сигнификаторы карты и выдаёт чёткий вердикт: ДА, НЕТ, ВОЗМОЖНО или НЕЯСНО
5. Сохраните результат в свой личный журнал вопросов

Это хорарная астрология — предсказательная техника с 600-летней историей, кодифицированная Уильямом Лилли в «Христианской Астрологии» (1647). Ваш вопрос анализируется по карте, построенной на точный момент его задания — не по солнечному гороскопу дня.

ЧТО ВЫ ПОЛУЧАЕТЕ

- Мгновенный вердикт: ДА / НЕТ / ВОЗМОЖНО / НЕЯСНО с уровнем достоверности (ВЫСОКАЯ / СРЕДНЯЯ / НИЗКАЯ)
- Интерпретация ИИ на понятном языке — без специальных знаний
- Планеты-сигнификаторы вопроса и спрашивающего: знак, дом, достоинство
- Сходящиеся аспекты между сигнификаторами — ключевые совершения вердикта
- Тайминг: примерное «когда» для благоприятных исходов
- Индикатор силы карты (радикальность 0–100) — насколько карта пригодна для суждения
- Предупреждение о Луне без курса с указанием исключений по Лилли
- Журнал вопросов: все расклады сохраняются на устройстве, сгруппированы по месяцам

ЧЕМ ЭТО ОТЛИЧАЕТСЯ

Каждое другое хорарное приложение показывает сырую карту и ожидает, что вы разберётесь сами. Hora интерпретирует её за вас — используя тот же классический свод правил (существенные достоинства, сигнификаторы домов, рецепция), которым пользуется обученный астролог.

Другие хорарные инструменты (Morinus, Janus, Solar Fire) работают только в браузере или на компьютере. Hora — первое красивое нативное мобильное приложение хорарной астрологии.

ЧТО ТАКОЕ ХОРАРНАЯ АСТРОЛОГИЯ

Хорарная астрология — это не солнечный гороскоп. Она не описывает вашу личность и не анализирует транзиты. Она отвечает на один конкретный вопрос, анализируя положение планет в момент его задания. Метод разработан Уильямом Лилли в Англии XVII века и остаётся самой строгой предсказательной техникой в традиционной западной астрологии.
```
