# App Icon Design Specification — Hora: Horary Chart

**Version:** 1.1
**Date:** 2026-06-05
**Output:** assets/images/icon.png (iOS + Android), assets/images/android-icon-foreground.png (Android adaptive), assets/images/android-icon-monochrome.png (Android notification)
**Generator:** scripts/generate-icon.js (Node.js + sharp)

---

## Design Intent

The icon communicates Hora's core visual identity in a single glance: a celestial, premium, dark theme with a zodiac clock face — the central motif of horary astrology, where a chart is cast for the exact moment a question is asked (*hora* = hour).

**Design principles:**
- Instantly recognizable at 60×60pt (small list view) AND 1024×1024px (store marketing)
- Premium dark background matches the app's Cosmos Dark theme
- Single clear focal element (clock face with pointer hand) — no text, no complexity
- Differentiates from common astrology apps (Co-Star uses circle/dot, The Pattern uses spiral, Nebula uses gradient sky); the clock motif uniquely signals "time of asking"

## Geometry

**Shape:** Zodiac clock face — circular dial with 12 tick marks, a pointer hand at ~10 o'clock (300° clockwise from 12), a 4-point star at the pointer tip, and a center pivot dot.

All coordinates are for a 1024×1024 canvas with center at (512, 512).

### Clock rings

| Ring | Radius | Stroke | Opacity | Purpose |
|---|---|---|---|---|
| Outer ring | 450 px | 15 px | 60% | Main dial boundary |
| Inner ring | 348 px | 6 px | 18% | Subtle inner track |

### Tick marks

| Type | Count | Inner radius | Outer radius | Stroke width | Opacity |
|---|---|---|---|---|---|
| Cardinal (N/E/S/W) | 4 | 174 px | 450 px | 18 px | 90% |
| Minor | 8 | 317 px | 450 px | 10 px | 40% |

Degrees (clockwise from 12 o'clock):
- Cardinal: 0°, 90°, 180°, 270°
- Minor: 30°, 60°, 120°, 150°, 210°, 240°, 300°, 330°

### Pointer hand

| Segment | Angle | Length (from center) | Stroke width | Opacity |
|---|---|---|---|---|
| Main hand | 300° (~10 o'clock) | 307 px | 22 px | 100% |
| Counterweight tail | 120° (opposite) | 92 px | 14 px | 50% |

### 4-point star at pointer tip

- Center: tip of the pointer hand (pt at 300°, r=307)
- Arm half-length: 56 px (forms diamond-style 4-point star via 8 vertices at ±S and ±S×0.26)
- Fill: gold

### Center pivot dot

| Element | Radius | Fill | Inner highlight |
|---|---|---|---|
| Outer dot | 36 px | Gold | — |
| Inner highlight | 15 px | White at 40% opacity | Simulates glass/dome |

### Radial glow

- Type: `radialGradient` centered at (50%, 50%), r = 50%
- Stop 1: `#8B5CF6` (violet) 15% opacity at center
- Stop 2: `#8B5CF6` 0% opacity at edge
- Applied as a large circle (r=496) behind the clock face

## Android Adaptive Icon

The adaptive foreground uses the same clock on a **transparent background**. The dark fill (`#070714`) is provided as the `backgroundColor` in `app.json`'s `android.adaptiveIcon.backgroundColor`.

This ensures the icon renders correctly in:
- Circle crop (Google Pixel launcher)
- Rounded square crop (Samsung, Xiaomi)
- Full bleed crop (some custom launchers)

---

## Android Monochrome Icon

`assets/images/android-icon-monochrome.png` renders the same clock geometry with all fills and strokes set to **solid white** on a transparent background. Android uses this for notification icons and themed (Material You) icon variants.

---

## Generation Script

Run: `npm run generate:icon`

This executes `scripts/generate-icon.js` which:
1. Computes all clock geometry (tick endpoints, pointer tip, star vertices) from angle/radius helpers
2. Assembles three SVG variants: opaque (icon), transparent (adaptive), white-on-transparent (monochrome)
3. Converts each to 1024×1024 PNG via `sharp`
4. Writes all three files to `assets/images/`

**Dependency:** `sharp` (devDependency — install once with `npm install --save-dev sharp`)

## EAS Build Icon Path Mapping

`app.json` references:
```json
"icon": "./assets/images/icon.png",
"android": {
  "adaptiveIcon": {
    "foregroundImage": "./assets/images/android-icon-foreground.png",
    "monochromeImage": "./assets/images/android-icon-monochrome.png",
    "backgroundColor": "#070714"
  }
}
```

All three output paths from the generator match these references exactly — no manual copy step needed.

## Quality Checklist (manual review before submission)

- [ ] Clock pointer reads clearly at 60×60pt (thumbnail view) — hand direction visible
- [ ] 4-point star at tip is distinguishable at small sizes
- [ ] No text in the icon (Apple rejects icons with text that duplicates the app name)
- [ ] No transparency in iOS icon (assets/images/icon.png must be fully opaque)
- [ ] Android adaptive icon: clock fits within central 66% safe zone
- [ ] Android monochrome: all white, no gold/violet (verify with `file` or image inspector)
- [ ] Colors match in-app design (background `#070714`, gold `#F5C842`, violet glow `#8B5CF6`)
- [ ] No other app on the store uses an identical or near-identical icon (verify visually)
